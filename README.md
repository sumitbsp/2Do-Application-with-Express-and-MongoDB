
# A 2Do APP

- Made with Express and MongoDB

- User Authentication and Authorization with Passport JS

**Features**

- User Sign Up
- User Sign In
- User Profile Uopdate
- Can add a todo with due date, category and priority
- High priority todos gets listed on top
